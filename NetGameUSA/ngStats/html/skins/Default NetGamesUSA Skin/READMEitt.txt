Questo è un backup della skin predefinita NetGames USA che è stata installata con Unreal Tournament. Se incontrate problemi con le skin che provate, potrete sempre utilizzare questa skin per tornare a una modalità operativa.

Copiate questa directory per intero, compresi il file "skin.css" e la directory "custom_graphics", accedete alla directory UnrealTournament\NetGamesUSA.com\ngStats\html\skins e sostituitene tutti i file.

Se incontrate dei problemi o avete bisogno di assistenza nell'utilizzo o nella creazione di skin personalizzate, visitate il sito di ngStats per Unreal Tournament all'indirizzo http://www.NetGamesUSA.com/ngStats/UT/
